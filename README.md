# next-nake-sass
> Sass for nake

## installation
```bash
npm install -S afeiship/next-nake-sass --registry=https://registry.npm.taobao.org
```

## usage
```js
import nxNakeSass from 'next-nake-sass';

nxNakeSass(['test/*.scss'], 'dist');
```
